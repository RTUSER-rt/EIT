S6
EIT Diagnosis — 最佳 PEEP 判讀
OD/CL crossing point + ΔEELI 趨勢分析
調整回原本的呼吸模式，開啟 EIT Diagnosis view 及 ΔEELI-trend view。
訊號判讀說明
橘色 C loss HP：PEEP 過高 → compliance ↓ → 過度膨潲（Overdistension）
白色 C loss LP：PEEP 過低 → compliance ↓ → 塌陷（Lung collapse）
交錯點（Crossing point）：最少 overdistension & 最少 collapse = 最佳 PEEP
在 ΔEELI-trend view 中，觀察 Decremental trial 階段 ΔEELI 變化量最大的 PEEP 值，即為 UIP。
最終 PEEP 應介於 AOP 與 UIP 之間，並確認 ΔEELI 在此 PEEP 維持穩定。
